# Zic-Pac-Toe

Zic-Pac-Toe es una versión temática de Plants vs. Zombies del clásico juego Tres en Raya (Tic-Tac-Toe), construida con Python y Pygame. Dos jugadores se turnan haciendo clic en las celdas del tablero: el **Girasol** representa al jugador X y el **Zombie** representa al jugador O. El turno inicial se asigna aleatoriamente al comenzar cada partida.

El objetivo es el mismo que el Tres en Raya tradicional: alinear tres de tus fichas en fila, columna o diagonal antes que tu oponente. Para jugar, simplemente ejecuta `python main.py` (requiere Python y Pygame instalados) y haz clic en las celdas del tablero para colocar tus fichas.
